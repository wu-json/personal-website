├── Heroes         — coming soon
└── Garden         — coming soon

## Local Development

This project uses [curse](https://github.com/wu-json/curse) for running local development processes.

```sh
# install deps
bun install

# run dev server
curse
```

## Adding a Fragment

Use the `/add-fragment` slash command in Claude Code to add a new photo album.
